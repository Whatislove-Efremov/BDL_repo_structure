# Task Requirements

## Lecture: Python for ingestion

### Homework Description
В этом домашнем задании студенты должны реализовать скрипт на Python, основанный строго на описании задачи.  
Цель состоит в том, чтобы закрепить концепции, рассмотренные в лекции, с помощью практических упражнений по программированию.
Задания разделяются на 2 части:
1. По 2 Небольшие задачи после каждой темы.
2. Полноценный ETL-скрипт, после прохождения всего раздела Python, для загрузки данных из CSV-файлов в базу данных.

### Requirements
- Решение должно в точности соответствовать формулировке задачи.
- Все необходимые переменные, вычисления и выходные данные должны быть реализованы.
- Сценарий должен выдавать ожидаемые результаты.
- Следует использовать только концепции, рассмотренные в текущей лекции.
- Если в задаче указаны дополнительные ограничения (например, конкретные операторы или конструкции), они должны соблюдаться.

### Submission Guidelines
- Отправьте один файл с расширением ".py".
- Скрипт должен выполняться без ошибок.
- Выходные данные должны соответствовать ожидаемому формату.
- Не включайте ненужные функции, которые не требуются для выполнения задачи.
