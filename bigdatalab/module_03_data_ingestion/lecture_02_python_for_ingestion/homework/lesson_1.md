# Задание: Обмен значений переменных и расчёт стоимости партии товара

## Бизнес-контекст

В системе складского учета обнаружена ошибка присвоения категорий: данные двух товарных позиций перепутаны.  
Необходимо произвести программную корректировку (обмен значений переменных) и рассчитать полную стоимость корректной товарной партии фруктов с учетом НДС для финансового отчета.

---

## Задача

- Инициализировать переменные с ошибочными значениями категорий.
- Выполнить обмен значений переменных `category_a` и `category_b`, используя синтаксис Python (без временной переменной), чтобы в `category_a` оказалось верное название.
- Рассчитать общую стоимость партии товара (`total_value`) для исправленной категории фруктов (`category_a`) по формуле:

`Стоимость партии = (Цена × Количество) + (Цена × Количество × Ставка НДС)`


- Вывести в консоль исправленную категорию и итоговую сумму.
- Снабдить код однострочными комментариями (`#`), поясняющими шаги вычислений.

---

## Входные данные (задать в коде)
```
- `category_a`: `"Vegetables"` — ошибочно присвоено фруктам  
- `category_b`: `"Fruits"` — ошибочно присвоено овощам  
- `price_per_unit_a`: `150` — цена за ящик партии фруктов  
- `quantity_a`: `40` — количество ящиков партии фруктов  
- `vat_rate`: `0.2` — НДС 20%
```
---

## Ожидаемый результат
```
Текущая категория A: Fruits
Общая стоимость партии с НДС: 7200.0
```
