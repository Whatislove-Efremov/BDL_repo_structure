# Задание: Логические проверки и операторы присваивания для товара

## Бизнес-контекст

Система EcoMarket подсвечивает товары как «Хит продаж» и принимает решения по отображению и пополнению товаров на основе бизнес-правил.  
Дополнительно показатели товара могут меняться в течение дня: цена повышается, складской остаток пересчитывается после поставки.  
Необходимо реализовать проверки с использованием операторов сравнения, логических операторов и расширенных операторов присваивания.

---

## Задача

- Создать исходные переменные товара.
- Рассчитать `is_hit` по правилу:  
  товар является хитом, если `price < 3` и товар от местного фермера.
- Вывести в консоль результат проверки:

`Является ли товар хитом? <True/False>`

- Добавить проверки (каждая проверка — отдельная переменная и вывод):

  - `has_supplier` — поставщик указан (проверка через `is not None`)
  - `can_show_in_app` — показываем товар в приложении, если есть в наличии и поставщик указан
  - `needs_restock` — пополнение нужно, если остаток `<= 20` или товар является хитом
  - `is_blocked` — товар не участвует в акции, если он не от местного фермера  
    (использовать `not (условие)`)

---

## Проверка приоритетов операторов and / or

- Реализовать правило скидки:

`(есть купон или есть карта) и сумма покупки > 50`

- Посчитать два варианта:
  - `discount_without_brackets` — без использования скобок
  - `discount_with_brackets` — с использованием скобок
- Вывести оба результата в консоль.

---

## Изменение значений с помощью расширенных операторов присваивания

- Увеличить `price` на `1.0` с помощью `+=`
- Увеличить `stock_quantity` в `2` раза с помощью `*=`
- Вычислить число полных коробок по `10` кг:
  - создать переменную `boxes = stock_quantity`
  - применить `boxes //= 10`

После изменений:
- повторно вычислить и вывести `is_hit`
- повторно вычислить и вывести `needs_restock`

## Подсказки

- Для проверки `None` используйте `is None` / `is not None`.
- Для отрицания условия используйте скобки: `not (условие)`.
- В сложных логических правилах используйте скобки для явного задания приоритета.
- Для изменения значений применяйте операторы `+=`, `*=`, `//=`.
